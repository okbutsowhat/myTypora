# 正则表达式Regular Expression

> 主要作用:匹配字符

练习工具: <http://regexr.com>

## "`.`" 匹配除断行外任意一个字符

转义字符可用在正则表达式, 例如: 想要匹配`3.14`, 则可以:

```
/3\.14/g
```

## 匹配字母和数字 `\w\W `
### ` \w ` 匹配阿拉伯数字, 引文大小写字母和下划线

例如:

```
/\w/g

12345678qwert
&(*&*)

==>
12345678qwert

```

### ` \W ` 匹配除被` \w `排除在外的字符(` \w `的补集)

## 匹配数字 ` \d\D `
` \d `: 仅仅匹配数字
` \D `: 除了数字都匹配

## 匹配空白字符 ` \s\S `
` \s `: 匹配空白字符:空格, 制表符, 断行等
` \S `: 匹配被` \s `排除在为的字符 ( ` \s `的补集)

## 字符集合 `[a-z A-Z 0-9]`
`[a-z]` 可以匹配a-z的所有字符, 若想匹配所有的中文字符可以使用:`[\u4e00-\u9fa5]`

> 注：\u是一种字符编码格式, Unicode, 万国码, 用数字代表文字的一项标准
> <http://graphemica.com> 万国码查询网站

## 重复一次或多次 `+ `
` [范围]+ `重复一次或多次, 例如:
```
/yo+/g

yo
yoooo

//两者都会匹配到
```
```
/[yY][oO]+/g

yo
yoooo
Yo
YOOOOO

//四者都会匹配到
```

## 重复零个和多个` * `
`[范围]*` 重复零次或多次, 可以有可以没有, 例如:
```
/[yY][oO]+[!~\.]*/g

yo
yooo
Yo
YOOOO
yo!
yo.
yoooo~~

//以上都能匹配到
```

## 重复零个或一个` ? `
`[范围]?`出现零次或一次，例如：
```
/[yY][oO]+p?[!~\.]*/g

yop！
yoooop
Yooop
yo
yooo
Yo
YOOOO
yo!
yo.
yoooo~~

//以上都能匹配到
```

## 指定重复范围 (重复a到b次)
```
/yo{1,4}/g

yo
yoo
yooo
yoooo
yooooooooo

==>
yo
yoo
yooo
yoooo
yoooo
```
`\b`加边界,
```
/yo{1,4}\b/g

yo
yoo
yooo
yoooo
yooooooooo

//最后一条不会选到
```

## 分组匹配
```
/(.+)飞飞,(.+)追追/g

李栓蛋飞飞,王花花追追

//会整句都匹配到, 并把`李栓蛋`和`王花花分别分组为`$1`和`$2`

$2飞飞,$1追追
==>
王花花飞飞,李栓蛋追追
```

## 分组匹配但不捕获
`?:`表示只匹配但是不捕获
```
/1(?:37|38|82|83)\d{4}(\d{4})/g
//匹配数字1,匹配37或38或82或83且不纳入分组,匹配数字四位,匹配数字四位且分组

13712241000
13822232000
18289993000
18398764000

13142531000
13211132000

$1==>
1000
2000
3000
4000
```

## 正向预查
### 正向肯定预查
`?=` **后面**需要有xx字符, 但不匹配他
例如:
```
/\d+(?=元)/g

100元
1000元
100刀
100磅

==>
100
1000
```
### 正向否定预查
`?!` **后面**有xx字符的不匹配，匹配没有xx字符的
例如：
```
/\d+(?！元|\d)/g

100元
1000元
100刀
100磅

==>
100
100
```

## 反向预查
### 反向肯定预查
`?<=` **前面**需要有xx字符, 但不匹配他
例如:
```
/(?<=¥)\d+/g

¥100
¥1000
＄100
£100

==>
100
1000
```
### 反向否定预查
`?!` **前面**有xx字符的不匹配，匹配没有xx字符的
例如：
```
/(?<!¥)\d+/g

¥100
¥1000
＄100
£100

==>
100
100
```
> js不支持反向预查

# 常用正则表达式匹配规则

## 1 限制输入为中文、英文、数字、常用符号
```js
oninput="value = value.replace(/[^\a-\z\A-\Z0-9\u4E00-\u9FA5\u0022\ \,\，\.\。\/\？\?\、\|\[\]\{\}\-\=\_\+\!\！\@\#\$\%\^\&\*\(\)\（\）\;\:\'\“\”\‘\’\`\~\·]/g,'')"
```

## 2 限制输入为中文、英文、数字

oninput="value =value.replace(/[^\a-\z\A-\Z0-9\u4E00-\u9FA5\ ]/g,'')"

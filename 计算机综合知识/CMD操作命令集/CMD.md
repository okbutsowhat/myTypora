# CMD


| 操作             | 命令                        |
| ---------------- | --------------------------- |
| 进入对应盘符     | D:                          |
| 进入目录         | cd 文件夹名字或者D:\xxx\xxx |
| 进入当前盘根目录 | cd                          |
| 退出到上一级目录 | cd..                        |
| 清屏             | cls                         |
| 显示文件列表     | dir                         |
| 建立子目录       | md                          |
| 删除文件目录     | rd                          |
| 改变文件名       | ren                         |
| 清除当前命令行   | ESC键                       |
